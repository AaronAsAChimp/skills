# Brainstorm

This is a skill that will help you solve big problems, think through issues, or come up with ideas. This is the process:

1. State the problem clearly
2. Use divergent thinking to generate many possible solutions
3. Use convergent thinking to weed out the worst ideas
4. State the solution

## Stating the problem

A good brainstorming session starts with stating the problem. Its important to be clear and unambiguous.

- Do not assume a cause or prescribe a solution
- Address the 5 W's: Who, What, When, Where, Why.
- Ask questions to clarify the problem.
- Gather evidence that shows this is a real problem that requires a solution.
- Note any edge cases

## Divergent thinking

Next, come up with possible solutions. There are no bad ideas. The goal is to widen your understanding of the problem and the potential solutions.

- Come up with many possible solutions
- Write out each idea and use it to inform your understanding 
- Continue coming up with solutions until you feel like there is nothing left to explore.

## Convergent thinking

This step is to start narrowing down the list of ideas you came up with in the previous step. Be critical of each idea:

- How well does it address the problem
- Does it have potential side effects
- Does it address all of the edge cases

Narrow the list down to the 3 best solutions.

## State the solution

In this final step present the user with your best solution. Also include the other two so they can make an informed decision.
